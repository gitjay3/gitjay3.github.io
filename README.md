# Portfolio

개인 포트폴리오 웹사이트

## Tech Stack

- React
- TypeScript
- Vite

## Getting Started

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 빌드
pnpm build
```

## Structure

```
src/
├── App.tsx      # 메인 컴포넌트 (프로필, 스킬, 프로젝트 데이터)
├── App.css      # 컴포넌트 스타일
├── index.css    # 글로벌 스타일
└── main.tsx     # 엔트리 포인트
```
